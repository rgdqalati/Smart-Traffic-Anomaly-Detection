# Smart Traffic Anomaly Detection

## Overview

This project presents a machine learning-based anomaly detection system designed to protect smart city traffic lights from slow data poisoning attacks. The solution utilizes an LSTM Autoencoder model to detect abnormal traffic behavior and improve the reliability of intelligent transportation systems.

## Technologies Used

- Python
- TensorFlow
- Keras
- SUMO
- OMNeT++
- Machine Learning
- Deep Learning
- LSTM Autoencoder

## Key Features

- Predictive anomaly detection
- Detection of slow data poisoning attacks
- Smart traffic simulation
- AI-driven traffic monitoring
